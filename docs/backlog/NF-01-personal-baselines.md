# NF-01 · «Сьогодні vs твоя норма» — довгострокові персональні базлайни

**Тип:** нова фіча (аудит 2026-07) · **Оцінка:** M (~2 тижні) · **Пріоритет:**
високий — перший серед нових (RICE 6.3) · **Залежності:** немає (дані вже в БД;
garth не потрібен)

## Суть

Метрики зараз трактуються проти короткого вікна (~21 день `get_recent_extra`);
Garmin дає baseline-band тільки по HRV. Нічний Python-джоб рахує по кожній
метриці rolling-персентилі (30/90/365 днів, сезонні) → компактний `norm`-знімок
у БД; ранковий звіт отримує «сьогодні / твій p50 / твоя смуга». LLM нічого не
рахує — тільки коментує готові відхилення. Прибирає головну скаргу на
конкурентів («метрики без контексту»); підсилює EP-08-пороги і NF-06.

## Вартість

+300–500 токенів input/день ≈ +$0.001–0.002/день.
