# NF-03 · Режим «хвороба / подорож» — ремонт плану одним тапом

**Тип:** нова фіча (аудит 2026-07) · **Оцінка:** M (~2 тижні) · **Пріоритет:**
середній (RICE 0.6 — рідко, але в момент спрацювання цінність максимальна;
ефект на довіру непропорційний score) · **Залежності:** EP-08-детектор
(автотригер) бажано; confirm-флоу `PlanEdit` ✅

## Суть

Жорсткі плани розсипаються від тижня грипу (підтверджений біль конкурентів);
EP-02 полегшує *день*, але «випав на 5 днів» — досі ручні `/plan`-едити.
Тригер з EP-08-детектора (HRV+RHR+температурний патерн хвороби) або команда
`/sick` → Claude отримує план + факт пропуску → пропонує *перебудову блоку*
(зсув довгого, скасування інтенсивності на тиждень, re-ramp по 10%-правилу)
через існуючий confirm-флоу `PlanEdit`. Формулювання консервативні, не медичні.

## Вартість

Рідкі виклики (1–3/квартал), Sonnet, контекст плану ≈ $0.05/випадок.
