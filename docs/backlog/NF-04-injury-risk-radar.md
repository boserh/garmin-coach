# NF-04 · Травматичний радар (injury-risk сигнали)

**Тип:** нова фіча (аудит 2026-07) · **Оцінка:** M (~2 тижні поверх EP-08+12) ·
**Пріоритет:** середньо-високий після EP-08/EP-12 (RICE 0.9; одна попереджена
травма окупає проєкт назавжди) · **Залежності:** EP-08 (детектор), EP-12 (болі)

**Перетини (ANALYSIS.md §1.6):** EP-08 ↔ EP-12 ↔ NF-04 — один детекторний шар
«сигнали ризику», різні джерела правил.

## Суть

ACWR вже є в контексті плану, але ніхто не дивиться на **комбінацію** ранніх
сигналів. Розширення EP-08-детектора load-side правилами: «ACWR >1.4 два тижні
поспіль», «той самий біль ≥2 рази за 14 днів» (з EP-12), «RPE росте при
стабільному темпі» → severity-скор → попередження + пропозиція deload через
EP-02-механіку. False positives вбивають довіру — «тихий» режим калібрування
2–3 тижні обов'язковий (як у EP-08).
