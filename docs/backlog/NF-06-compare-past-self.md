# NF-06 · «Я-минулорічний» — порівняння з минулим собою

**Тип:** нова фіча (аудит 2026-07) · **Оцінка:** M (~1.5 тижні) · **Пріоритет:**
середній (RICE 2.0; сильний retention-гачок) · **Залежності:** немає (глибока
історія вже в БД після GDPR-бекфілу)

## Суть

«Я зараз швидший, ніж перед торішнім півмарафоном?» — на це не відповідає ніхто
з конкурентів. Python збирає зіставні вікна (той самий місяць рік тому /
N тижнів до минулого старту): обсяг, темп easy-пробіжок, HRV-базлайн, VO2max,
вага довгих → один Sonnet-виклик наративізує. Команда `/compare [період]` +
автоблок у дайджесті раз на місяць. Головне — чесність порівнянь: різні
сезони/умови явно зазначати в промпті.
